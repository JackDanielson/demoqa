# demoqa
Automated Cypress Test Suite for www.demoqa.com

## Overview
Automated E2E tests for www.demoqa.com as requested, using Cypress with JavaScript.

## Technologies Used:
- Cypress
- JavaScript
- Mochawesome Reporter

## Setup and Execution
### 1. Clone Repository:

`git clone https://github.com/JackDanielson/demoqa.git`
`cd demoqa`

### 2. Install Dependencies needed:

`npm install`

### 3. Run Tests:

`npx cypress run` - Headless mode
`npx cypress open` - Interactive mode

## Reports

Reports are going to be generated in the /cypress/reports folder after execution.
